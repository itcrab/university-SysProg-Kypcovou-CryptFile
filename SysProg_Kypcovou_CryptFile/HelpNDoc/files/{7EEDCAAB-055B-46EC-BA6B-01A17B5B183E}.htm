Работа с окном - пользовательские данные.

В этом окне вам предлагается ввести следущие параметры:

Параметр

Значения

Описание




Тип работы




Данный параметр указывает МАСТЕРУ шифрования файлов, какую работу ему нужно будет выполнить.


Шифрование - вы будите шифровать выбранный файл.

Дешифрование - вы будите дешифровать указанный файл.








Алгоритм









Данный параметр указывает МАСТЕРУ шифрования файлов каким алгоритмом необходимо шифровать/дешифровать выбранный файл.


Перечень алгоритмов:

DES - неплохой и быстрый алгоритм шифрования данных.

TripleDES - тоже самое, что и DES, только с тремя перепроходами. Более надежный нежели DES, но и более ресурсоемкий.

RC2 - более быстрый нежели DES алгоритм при относительно одинаковой стойкости к взлому.

Rijndael (AES) - наиболее предпочтителен с точки зрения стойкости к взлому.



Путь к файлу



текстовое поле

Сюда вам неоходимо вставить путь до выбранного файла.

Можно либо вписать его "руками", т.е. с нуля, либо выбрать необходимый файл с помощью обзора (кнопка "...").

Пароль доступа

закрытое текстовое поле

Пароль для доступа к файлу (необходим при дешифровки выбранного файла). Указывается при шифровании/дешифровании.







Имя выходного файла







текстовое поле

Сюда вам неоходимо вставить путь до выходного файла.

Можно либо вписать его "руками", т.е. с нуля, либо выбрать необходимый файл с помощью обзора (кнопка "...").


Если данное поле не будет заполнено, то будет использовано стандартное имя файла:

  • если шифрование, то <путь до файла + имя файла>.<алгоритм>. Пример, 1.txt.DES
  • если дешифруется, то <путь до файла + имя файла>.<алгоритм + слово "none">. Пример, 1.txt.DES.DESnone


После заполнения всех необходимых данных - нажмите кнопку "Далее" - после чего начнется Шаг 2 - процесс шифрования/дешифрования.
