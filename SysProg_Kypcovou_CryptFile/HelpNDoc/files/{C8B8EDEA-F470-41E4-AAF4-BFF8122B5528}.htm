Окно - процесс шифрования/дешифрования


Работа с окном - процесс шифрования/дешифрования.

В этом окне отображается вся информация о ходе процесса шифрования/дешифрования.


Для этого использованы следущие элементы:

  • Лог - показывает текстовую информацию о ходе процесса шифрования/дешифрования.
  • Процесс - индикатор - показывает процентное соотношение текущей работы по шифрованию/дешифрованию выбранного вами файла.


Кнопка "Повтор" служит для того, чтобы еще раз запустить мастер шифрования  фалов для шифрования/дешифрования файла.
